## How Does Proppr Calculate Value?

The proprietary "Cebro" statistical model (NOT AI) calculates fair odds using:

```
Value % = ((Bookmaker Odds ÷ Model Odds) - 1) × 100
```

When bookmaker odds exceed model odds, you have positive expected value (+EV).
